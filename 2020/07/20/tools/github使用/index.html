#Pull Request

是一种代码合作开发方式，将修改过的代码合并到主干的一种流程。这个过程中，还可以进行讨论、审核和修改代码。

  • fork是指克隆别人的代码到自己的仓库
  • Base 是你希望提交变更的目标
  • Head 是目前包含你的变更的那个分支或仓库
