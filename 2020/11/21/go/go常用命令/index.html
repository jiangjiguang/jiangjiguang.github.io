go常用命令

发表于 2020-11-21 更新于 2021-06-20 分类于 go 阅读次数：

命令概览

go test

  • 常用命令

go get

  • 借助于代码管理工具远程拉取代码包以及依赖包，并自动完成编译和安装。

  • 使用前需要安装与远程包匹配的代码管理工具，如Git

  • 实际执行时分为两步：下载源码包，执行go install

  • 参数介绍

    -d 只下载不安装

    -t 同时也下载需要为运行测试所需要的包

    -u 强制使用网络去更新包和它的依赖包

    -v 显示执行的命令

  • 使用案列

go install

  • 与 go build命令类似，并且附加参数绝大多数相同

  • 只是将编译的中间文件放在GOPATH的pkg目录下，以及将编译结果放到GOPATH的bin目录下

  • 使用案例

go mod

  • 如果使用到了第三方的包，那么就需要go.mod文件

  • 在该模式下，可以在任何路径下写代码（不需要在GOPATH路径）
