工具实践-sar

发表于 2020-09-09 分类于 linux 阅读次数：

说明

  • 全称System Activity Reporter 系统活动情况报告，是linux上最为全面的系统性能分析工具之一

    可以从多方面对系统活动进行报告：

    1、文件的读写情况

    2、系统调用的使用情况

    3、磁盘I/O

    4、CPU效率

    5、内存使用状况

    6、进程活动以及IPC有关的活动

  • 具体使用

使用实践

  • CPU资源监控

  • 进程等待运行队列和平均负载状态监控

  • 内存和交换空间监控

  • 内存分页监控

    ​

  • inode、文件和其他内核表监控
