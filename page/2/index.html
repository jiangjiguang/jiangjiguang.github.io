Zeppelin基本知识

  • Paragraph：代码运行、开发的最小单元

    1、%开头，指定Interpreter

  • Node：Paragraph的集合

  • Notebook：Node的集合

  • Interpreter：解释器，对应一个引擎或一种语言

  • Interpreter Setting：Interpreter的配置信息

Zeppelin配置

  • hadoop classpath：flink会把该命令的输出结果放到classpath中

Interpreter binding mode

  • Shared：flink任务提交到同一个集群

  • Scoped

  • Isolated：分为两种 Per Note 和 Per User

    Per Note：每个Note都会创建一个新的flink cluster session

    Per User：每个用户都会创建一个新的flink cluster session

用户登录

  • 开启用户登录，具体配置看官网

Inline Configuration

  • 对flink Interpreter 进行定制化的配置

  • 需要点击右上角的 Interpreter binding 进行重启，properties格式

Scala基本知识

发表于 2021-03-04 更新于 2021-06-20 分类于 Scala 阅读次数：

基本知识

  • 面向对象（OOP）、函数式编程（FP）

  • 六大特性

    1、无缝Java互操作：Java和Scala堆栈可以自由混合以实现完全无缝的继承

    2、类型推断：

    3、并发与分布式：Actor

    4、特质：接口+抽象类，

    5、模式匹配：不仅可以配置值，还可以匹配类型

    6、高阶函数：函数式一等公民，可以用在任何地方，函数的参数可以是函数

  • 数据类型

    Unit：表示无值，等同void

    Null：空值或空引用

    Nothing：所有其他类型的子类型，表示没有值

    Any：所有类型的超类，任何实例都属于Any类型

    AnyRef：所有引用类型的超类

    AnyVal：所有值类型的超类

    Option类的两个子类：None、Some用于安全的函数返回值

    Nil：表示长度为0的list

  • var修饰变量，val修饰常量

  • 伴生类伴生对象：如果在同一个文件中，object和class名字相同时，可以互相访问私有属性

  • object不可以传参数，不能new，Trait不可以传参数，class需要使用new

  • to until

函数

  • 匿名函数

    1、可以将匿名函数返回给val定义的值

    2、匿名函数不能显示声明函数的返回类型

  • 偏应用函数

  • 高阶函数

    1、函数的参数是函数

    2、函数的返回类型是函数

    3、函数的参数和函数的返回类型是函数的函数

  • 颗粒化函数
